# bitcoin-price-tracker

Pulls the current BTC/USD spot price from a public API and prints it. Tiny — just enough to drop into a shell prompt or a cron job.

## Usage

    python3 btc.py

Outputs something like:

    BTC/USD: 67,432.18

## Why

I check this too often. Now my terminal does it for me.

## Dependencies

Just `requests`. Install with:

    pip install requests
